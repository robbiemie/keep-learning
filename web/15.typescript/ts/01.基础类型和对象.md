# 基础类型和对象类型

## 基础类型

> boolean、number、string、void、undefined、symbol、null

```ts
// 基础类型
const count: number = 123
const name: string = 'robbie'
```

## 对象类型

{}，Class，function,[]

```ts
// 对象类型
class Person {}

const person: {
    name: string
    age: number
} = {
    name: 'robbie',
    age: 12
}

const numbers: number[] = [1,2,3]

const getTotal: () => number = () => {
    return 100
}
```
